# Ejemplos de Simulacros de Entrevista Consular

Esta carpeta contiene ejemplos de transcripciones de simulacros de entrevista consular para diferentes tipos de visa.

## Estructura

Cada tipo de visa tiene dos ejemplos:
- **nervioso**: Solicitante que muestra signos de nerviosismo, respuestas vacilantes
- **seguro**: Solicitante que responde con confianza y claridad

## Tipos de Visa

1. **Visa de Vivienda** (Residencia)
   - `visa_vivienda_nervioso.txt`
   - `visa_vivienda_seguro.txt`

2. **Visa de Trabajo**
   - `visa_trabajo_nervioso.txt`
   - `visa_trabajo_seguro.txt`

3. **Visa de Estudio**
   - `visa_estudio_nervioso.txt`
   - `visa_estudio_seguro.txt`

## Uso

Estos ejemplos pueden usarse para:
- Entrenar el modelo de IA para generar recomendaciones
- Mostrar a los clientes ejemplos de buenas y malas prácticas
- Testing del sistema de análisis de simulacros
