# Shared Directory

Este directorio contiene codigo compartido y reutilizable entre features.

## Subcarpetas

### `/components`
Componentes UI reutilizables (Design System)

### `/hooks`
Custom hooks genericos

### `/services`
Servicios API base y utilidades HTTP

### `/utils`
Funciones utilitarias y helpers

### `/constants`
Constantes globales

### `/types`
Definiciones de tipos (para TypeScript o JSDoc)

### `/styles`
Estilos globales y variables CSS
